# Lab 1: Setup The Environment

## Objective
The objective of this lab is to prepare and configure the development environment required for machine learning activities, including Python, Anaconda, and Visual Studio Code.

## Tasks Completed
1. **Anaconda Installation:** Installed Anaconda Navigator and successfully configured the base environment.
2. **VS Code Configuration:** Connected Visual Studio Code to the Anaconda Python interpreter.
3. **Library Verification:** Verified the installation of essential libraries, including `pandas`, `numpy`, and `matplotlib`.
4. **Git Setup:** Initialized the Git repository and configured user credentials.
